# Changelog

All notable changes to this project will be documented in this file.

## [1.0.3] - 2026-03-13
- Initial packaged release "1.0.3".
- Features:
  - `StatusEffectLogic` — отображение и управление эффектами статуса.
  - `GroundEffectLogic` — поддержка визуализации эффектов на земле.
  - `MonsterLineLogic` — вспомогательная логика для отображения линий/маркировки монстров.
- Build: проект целится в `net8.0`.

## [1.0.4] - 2026-03-13
- Fix: make config file paths relative to the plugin directory (use AppContext.BaseDirectory).
- Build: updated and packaged plugin DLL for release v1.0.4.
