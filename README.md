Tracker
=======

Tracker — плагин для GameHelper, предназначенный для отображения эффектов статуса, земельных эффектов и подсказок по монстрам в игре.

Ключевые возможности
- Отслеживание и отображение эффектов статуса
- Логика отображения Ground Effects
- Отображение линий/маркировки для монстров

Сборка
1. Откройте [Tracker/Tracker.sln](Tracker/Tracker.sln) в Visual Studio или запустите `dotnet build` в папке `Tracker`.
2. Бинарники будут размещены в `Tracker/bin/Debug/net8.0/`.
3. После сборки файлы плагина копируются в `GameHelper/bin/Debug/net8.0/Plugins/Tracker/` (см. цель `CopyFiles` в `Tracker.csproj`).

Установка
1. Скопируйте содержимое папки `Tracker/bin/Debug/net8.0/` в папку плагинов GameHelper или используйте готовый релиз с GitHub Releases.
